## 🛡️ Privacy and Safety

This tool uses a local-first design. We store all your configuration files directly on your machine. The app does not send your data to external servers. It does not contain tracking scripts. Your private API keys and session tokens remain on your hard drive. 

## 🖇️ How It Works

Managing multiple Claude Code or Codex profiles often causes issues with login tokens. Apivot solves this by creating isolated environments. 

### Create a Profile
When you open Apivot, you see a list of profiles. Click the plus button to add a new setup. Give the profile a name. Enter your API credentials or link your account as you would for the standard coding interface.

### Switch Profiles
Select a profile from the main list. Click the Activate button. Apivot swaps your underlying system configuration files. You can now use your preferred coding tool with the new account settings instantly.

### Backup and Restore
You can save your current setup as a file. Use the Backup function to export your configuration. If you ever need to reset your environment, use the Restore button. Pick your backup file, and the application returns your settings to that state.

## ❓ Frequently Asked Questions

### Does this app change my main account login?
No. Apivot works by pointing your software to different local folders for configuration data. Your official account status remains unchanged on the service provider's servers.

### Can I delete a profile?
Yes. Select the profile in the list and click the trash icon. This removes the local configuration files. It does not delete your actual account on the cloud service.

### Why do I need this tool?
If you work with multiple clients or use different account permissions for your projects, manual switching is tedious. Apivot automates this process. It prevents accidental errors that might lock you out of your official environment.

### Will the app update itself?
The application checks for new releases when you start it. If an update exists, it shows a notification. You can then download the latest version to keep your tool running well.

## 🛠️ Troubleshooting

If you experience problems, follow these tips:

* Permission Errors: Run the application as an administrator if your system files are in a protected folder.
* Missing Profiles: Ensure you saved your backup files in a location where your user account has read access.
* Connection Issues: Check that your firewall allows Apivot to access the necessary API endpoints.
* Application Fails to Launch: Uninstall the app, restart your computer, and reinstall it from the latest release link at the top of this page.
